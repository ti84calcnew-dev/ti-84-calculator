The SAT has two math sections. One allows you to use a calculator; the other does not. The Ti 84 calculator  is permitted on the calculator allowed section, and is on the College Board's approved calculator list. It is one of the most used calculators in the SAT test centers.

Using the TI-84 well on the SAT is not just a question of knowing the buttons - it is about knowing when and how to use it to answer questions faster than working by hand. Here are eight specific techniques that make all the difference on test day.

Trick 1 - Graph Both Sides to Find the Answer
If a question asks you to solve an equation, you can eliminate all algebra and just graph the two sides of the equation. Put the left side of it on Y1 graph and the right side of it on Y2. The intersection of the two graphs is the answer; the x - value at the intersection is your answer. This is useful when the equation is difficult to solve manually. Press 2nd+TRACE to get into the CALC menu, select "intersect" and follow the prompts.

Trick 2 - Using the TABLE to Test Answer Choices
To open the TABLE, press 2nd + GRAPH. If you have a function typed in Y= then the table displays input and output side by side. When there are answer choices for a question, plug the answers into the table and make sure it gets the proper output. This avoids the necessity to solve the equation to any extent. To change the values displayed on the table, press 2nd + WINDOW to open the Table Setup menu and configure your values.

Trick 3 — Store Values with STO→
When there is a question where a certain value is used several times, save the value in a variable so you don't have to retype the value. Type the number and press the STO-> key then press ALPHA and a letter key, and then press the enter key. That letter now represents your worth throughout the session. For example, if you are using a problem that uses pi = 3.14159 and you need it in several calculations, store it in A and use A in every formula.

Trick 4 - Find Zeros without the Quadratic Formula
To find the equation where a quadratic is equal to zero, put it on a graph on Y= and use the zero finder. Press 2nd + TRACE to open CALC, choose option 2, which is "zero." The calculator requests a left bound, a right bound, and a guess. Move the cursor to either side of where the graph crosses the x- axis and press ENTER for each of the prompts. The calculator provides the zero to the exact. This is quicker and less prone to mistakes than applying the quadratic formula under time pressure.

Trick 5 - Trig Tricks to Use on SAT Questions
Always make sure your MODE is set to DEGREE before any trig question, unless the question tells you otherwise (i.e., if it states radians). Press mode and verify that the degree option is highlighted. For the questions based on the sin, cos or tan of certain angles, the calculator provides decimal values. If the answer choices are in fraction or radical form, be aware of common values: sin(30deg) = 0.5, cos(45deg) = 0.707, sin(60deg) = 0.866.
